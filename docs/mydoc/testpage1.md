✅ 详细解释 `<div>` 是什么？

`<div>` 是 HTML 中的块级元素（block-level element） 它本身没有语义含义
（不像 `<section>`, `<article>` 那样有明确意义）

常用于：

* 包裹一组内容
* 作为布局容器使用 
* 给这一组内容统一加样式或结构控制
